# 上海市民政局 · 适老化多商城系统 Demo

> 政府购买养老服务数字化转型试点 · 适老化产品多商城平台

🔗 **在线 Demo**：https://hongyehe-arch.github.io/xingfu-qiaomen-demo/elderly-mall/

## 项目简介

本项目是**上海市民政局**委托开发的「适老化产品多商城系统」前端 Demo，服务于老年人辅助器具/康复器械/适老化产品的在线选购。系统覆盖五大商城、整合政府补贴券自动抵扣、全流程语音引导，为老年用户提供极简的购物体验。

## 核心功能

### 🧧 五大商城
| 商城 | 功能 | 适用人群 |
|------|------|---------|
| 补贴商城 | 政府补贴券自动抵扣购买辅具 | 符合条件的老年人（低保/残疾/长护险） |
| 集采商城 | 社区/机构批量采购辅具 | 社区服务站、养老机构 |
| 租赁商城 | 按日/月/季租赁康复器械 | 短期康复、尝试使用 |
| 自费服务 | 上门护理/康复理疗/陪诊 | 所有老年人 |
| 自费硬件 | 智能血压计/跌倒手环/智能药盒 | 所有老年人 |

### 🔄 补贴三步流程
1. **资格认证**（一次性，约60秒）：微信授权 → 人脸识别 → 自动五维核验（公安/民政/残联/长护险/评估）
2. **领取补贴券**（每期，约10秒）：点击领取 → 券到账户 → 语音播报「¥300已到账」
3. **下单使用**（每次，约60秒）：浏览商品 → 券自动抵扣 → 微信支付差额

### 📊 9状态订单状态机
```
待支付 → 已支付 → 配送中 → 已送达 → 安装中 → 已完成
                                    ↓        ↓        ↓
                                 已取消   退货中 → 已退款
```

### 🔈 全流程语音播报
- 认证引导、补贴领取、支付确认均有语音提示
- 人脸识别时语音引导「请看着屏幕眨一下眼睛」
- 大字体 + 高对比度 + 语音三通道确保老人无障碍使用

### ⭐ 积分体系
- 领补贴券 +50 积分
- 补贴商城消费每¥10 +1 积分
- 自费商城消费 3倍积分
- 100积分 = ¥1 抵扣

## 适老化设计

| 设计原则 | 具体实现 |
|---------|---------|
| 大字体 | 身体文本 ≥ 18px，标题 ≥ 24px，价格 48px |
| 高对比度 | 白底黑字，彩色仅用于强调 |
| 大触摸区域 | 按钮最小高度 44px |
| 极简操作 | 补贴流程仅3步，每步1个按钮 |
| 全语音 | Web Speech API 中文播报，语速 0.85x |
| 政府标识 | 醒目政府徽章降低信任门槛 |

## 技术栈

- **前端**：纯 HTML5 + CSS3 + Vanilla JavaScript（零依赖）
- **持久化**：localStorage（模拟后端）
- **语音**：Web Speech API（SpeechSynthesis）
- **部署**：GitHub Pages（静态托管）

## 本地运行

```bash
python3 -m http.server 8765
# 访问 http://localhost:8765
```

## Demo 使用流程

1. 打开页面 → 默认进入「补贴商城」
2. 点击「开始认证」→ 走完3步认证流程
3. 点击「领取补贴券 ¥300」→ 券到账 + 语音播报
4. 选择商品 → 确认订单 → 补贴自动抵扣 → 支付
5. 查看「我的订单」→ 观察订单状态自动推进
6. 切换商城标签页体验其他商城
